Date : 10/03/2021

Author : CLEMENT Alexis

Description : Le tower defense est un type de jeu vidéo où l’objectif est de défendre une zone (ici des bâtiments) contre des vagues successives d’ennemis se déplaçant, en construisant progressivement des tours défensives.

Contrôles : Sélectionner une tourelle :	- & : Tourelle longue portée
					- é : Tourelle courte portée (tire plusieurs balles)
					- " : Tourelle missile (explosion à l'impact)
					- ' : Tourelle mur (ne tire pas mais permet de bloquer)

	- Clic gauche: Poser la tourelle sur un cellule adjacente à celle du joueur
	- Clic droit: Se déplacer
	- Echap: Quitter le jeu

Fonctionnalités: Pour gagner, le joueur doit :
	- Défendre les bâtiments constituant sa base
	- Déployer des tourelles pour éliminer les ennemis
